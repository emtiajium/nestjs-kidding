## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Pre-requisite
Node v14 LTS

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
